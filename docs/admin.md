---

## Admin Responsibilities

Admins are responsible for maintaining accurate and up-to-date menu data.

---

## Admin Capabilities

Once approved, admins can:

- Add menus for specific dates
- Edit existing menus
- Delete outdated or incorrect entries
- Manage meal categories:
  - Breakfast
  - Lunch
  - Snacks
  - Dinner

---

## Data Integrity Rules

Admins should ensure:
- No duplicate entries for the same date
- Proper formatting of meals
- Correct categorization

---

## Abuse & Revocation

Admin access may be revoked if:
- Data is intentionally corrupted
- Unauthorized changes are made
- Platform rules are violated

---

## Summary

Admin access exists to:
- Keep MenU reliable
- Ensure users trust the data they see

With access comes responsibility.
